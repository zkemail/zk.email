Built for inboxes that never stop.

Set rules once. Keyword Forwarder keeps checking on your schedule and tags processed threads so nothing gets forwarded twice — and without asking the destination inbox to approve anything.

No recipient confirmation Gmail’s native forward requires the other mailbox to confirm. This add-on sends the message for you, so that step never happens.
Multiple automations Separate keyword/search → destination rules, each with its own stats and enable switch.
Live match preview See the latest emails a new filter would forward before you save it.
Your cadence Check every 5 minutes, every hour, or once a day — pick what fits the urgency.
Gmail search power Plain keywords or full Gmail queries like from:alerts@bank.com.
No double-sends Processed threads get a private label so reruns stay idempotent.
