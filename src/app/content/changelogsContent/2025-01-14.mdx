## ZK Email Updates

#### Registry
- Improved email handling with fixes for non-ASCII characters and base64 string rendering
- Enhanced blueprint management with confirmation modals and compilation state handling
- Added selector search parameters for archive API integration

Explore the Registry at [registry.zk.email](https://registry.zk.email/).

#### zkemail.rs
- Refactored email and regex input handling in generator module
- Implemented public key fetching from Google DNS and archive systems
- Added continuous integration pipeline and development guidelines

#### SP1 Implementation

- Updated implementation to SP1 v4

Checkout the [SP1 implementation repository](https://github.com/zkemail/sp1-zkEmail) for more details.

#### zkemail.nr

- Enhanced cryptographic operations by implementing REDC manipulation
- Conducted security audits addressing header validation, hashing elements, and limb normalization
- Updated JavaScript SDK to utilize aztec/bb.js library

Checkout the [zkemail.nr repository](https://github.com/zkemail/zkemail.nr) for more details.
