## ZK Email Updates

#### Registry
- Improved email handling with [fixes for non-ASCII characters](https://github.com/zkemail/registry/commit/67aa30031d22823bf854ff54ad5460cbeef99e52) and [base64 string rendering](https://github.com/zkemail/registry/commit/58a8a0504c184f8b7ac4b4b3e0ce77bddc08ea82)
- Added [confirmation modals](https://github.com/zkemail/registry/commit/e9fc715b47cce10630671cf82c34de2b2bd86597) and compilation state handling on the blueprint management
- Added [selector search parameters](https://github.com/zkemail/registry/commit/37cf53260d4bcce2489a4c42b4454cd9f963374b) for archive API integration

Explore the Registry at [registry.zk.email](https://registry.zk.email/).

#### zkemail.rs
- Implemented [public key fetching](https://github.com/zkemail/zkemail.rs/commit/8a940974e5bbb10951b18c0ce64b5b0faf300e31) from Google DNS and archive systems
- Added [continuous integration pipeline](https://github.com/zkemail/zkemail.rs/commit/6a97e627aa59c80c8780f1c42307c39d6f9c2aee) and [coding guidelines](https://github.com/zkemail/zkemail.rs/commit/37167fa463227be6f21e20298ad9d3441832c778)

#### SP1 Implementation

- Updated implementation to [SP1 v4](https://github.com/zkemail/sp1-zkEmail/commit/f60a7ca0315a9f7852b2566ac0c0711b32ff221d)

#### zkemail.nr

- Conducted security audits addressing header validation, hashing elements, and limb normalization
- [Updated JavaScript SDK](https://github.com/zkemail/zkemail.nr/commit/53b6798609bb5e0f1f8d2cd47a46f714f5de3464) to utilize aztec/bb.js library

Checkout the [zkemail.nr repository](https://github.com/zkemail/zkemail.nr) for more details.
