## ZK Email Updates

#### Registry
- Updated the version card buttons and information banner.
- Improved [blueprint creation](https://registry.zk.email/create/new) flow with field extraction UI.

Explore the Registry at [registry.zk.email](https://registry.zk.email/).

#### Archive
- Added filtering by date and domain for [email uploads](https://archive.prove.email/upload_gmail).

#### ZK Email Verify
- Implemented [SHA precompute for quoted-printable encoded selectors](https://github.com/zkemail/zk-email-verify/pull/240).

#### Email Transaction Builder
- Added GPU prover, improving ZK proof generation performance.

Learn more about the Email Transaction Builder on [GitHub](https://github.com/zkemail/email-tx-builder).

#### JWT Transaction Builder
- Added Decoded JWT, generated proof preview to the [JWT demo](https://jwt.zk.email/).
- Added circuit config to the verifier and auth and updated dependencies and documentation.

Learn more about the JWT Transaction Builder on [GitHub](https://github.com/zkemail/jwt-tx-builder).

#### SP1 Implementation

- Implemented basic email verification using the [zkemail.rs](https://github.com/zkemail/zkemail.rs/tree/feat/core-impl) library compatible with SP1.

Checkout the [SP1 implementation repository](https://github.com/zkemail/sp1-zkEmail) for more details.
